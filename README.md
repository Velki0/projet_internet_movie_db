# Projet Individuel JPA
Projet individuel pour ma formation CDA sous la supervision de Diginamic

# Sujet choisi : 
Projet Internet Movie DataBase

# Objectif
- Réaliser un dossier de conception (UML et modèle physique de données)
- Mettre en place la base de données relationnelle
- Mettre au point une application permettant d'initialiser la base de données avec des données existantes.
- Réaliser une seconde application dotée d'un menu et permettant à l'utilisateur de réaliser des recherches dans les données.

# Exigences 
- Qualité de code : Renseigner la Javadoc
- Les données suivantes sont des entités : Lieu de naissance, pays, langue, genre.
- Les lieux de naissance doivent être uniques
- Les pays doivent être uniques
- Les langues doivent être uniques
- Les genres doivent être uniques
- Les dates de naissance doivent être de type DATE en base de données
